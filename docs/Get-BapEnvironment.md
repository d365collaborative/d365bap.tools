# Get-BapEnvironment

## SYNOPSIS
Get environment info

## SYNTAX

```
Get-BapEnvironment [[-EnvironmentId] <String>] [-AsExcelOutput] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION
Enables the user to query and validate all environments that are available from inside PPAC

It utilizes the "https://api.bap.microsoft.com" REST API

Get-BapEnvironment
